LICZEBNOŚCI WYBRANYCH ZWIERZĄT I ROŚLIN

Liczba obserwacji: 21

Atrybuty (kolumny):
1. rok (ang. year)
2. zające (ang. hares)
3. rysie (ang. lynxes)
4. marchew (ang. carrot)

Kolumny oddzielone są znakami tabulacji.

Pierwszy wiersz zawiera nazwy kolumn w języku angielskim poprzedzone znakiem
komentarza (#).
